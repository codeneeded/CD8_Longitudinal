10. TECHNICAL NOTES
--------------------------------------------------------------------------------

10a. ADT export: Seurat v5's AverageExpression() applies expm1() to the
     data slot, assuming log-normalization. For DSB-normalized data this
     produces exponentially inflated values (e.g., CD36 avg = 10,941
     instead of 0.7 DSB). The pipeline uses manual rowMeans() from the
     data slot for all ADT exports.

10b. ADT scaling: scale_01() applied directly to DSB averages (no log10
     transform). DSB values are already on a meaningful scale.

10c. PctExpr files: clusters as rows, markers as columns (transposed
     from AvgExpr files).
